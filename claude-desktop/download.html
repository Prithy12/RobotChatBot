Installation Instructions

  1. Windows Installer: Download and run the installer. Follow the on-screen instructions to install the application.
  2. Windows Portable: Download the portable executable, and you can run it directly without installation. Perfect for use on USB drives or when you don't have installation permissions.

After Installation

  1. Launch the application by clicking the desktop icon or start menu shortcut.
  2. When prompted, enter your Claude API key. You can get an API key from Anthropic's Console.
  3. Start chatting with Claude!

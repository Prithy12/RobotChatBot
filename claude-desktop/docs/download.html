Running from Source Code

  1. Download the ZIP file using the button above.
  2. Extract the ZIP file to a folder on your computer.
  3. Navigate to either environment:
    • For production: go to environments/prod/ folder
    • For development: go to environments/dev/ folder
  4. Run the start.cmd file in that directory. It will:
    • Install all required dependencies
    • Build the application automatically
    • Launch the application
  5. When prompted, enter your Claude API key. You can get an API key from Anthropic's Console.

System Requirements

  • Operating System: Windows 10/11, macOS 10.13+, or Linux
  • Node.js: The start script will install all dependencies, including Node.js if needed
  • Disk Space: At least 500MB free space
  • Internet Connection: Required for API communication
